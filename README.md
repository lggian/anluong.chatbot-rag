# Chatbot based on RAG-web-ui
Using template free
```
https://github.com/rag-web-ui/rag-web-ui/tree/main
```
------------------------------------------------------
How to use this code:
1. Create (.venv) for ollama_service folder.

2. Import -r requirement.txt of ollama_service folder:
```bash
pip install -r requirements.txt
```

3.  Run main.py in the ollama_service folder:
```bash
python main.py
```

3. Create a new (.venv) for the rag_service folder. You need to run two (.venv) files in a way that prevents them from getting mixed up!
If it is mixed up, you will create (.venv) from step1 and step3 again (wasted your time).

4. Import -r requirement.txt of the rag_service folder:
```bash
pip install -r requirements.txt
```

5.  Run main.py in the rag_service folder: (It often has a bug at this line if you are wrong at step3, which I had warned about earlier)
```bash
python main.py
```
